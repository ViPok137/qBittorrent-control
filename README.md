# qBittorrent-control

Автоматизированный скрипт для умной очистки и управления загрузками в qBittorrent. Скрипт анализирует активность торрентов и удаляет ненужные, помогая поддерживать порядок в клиенте и экономить место на диске.

## Основные возможности

* **Удаление по неактивности**: Автоматическое удаление торрентов старше `X` дней, если их рейтинг (Ratio) не вырос, а количество личей (скачивающих) минимально.
* **Мгновенная чистка ошибок**: Скрипт сразу удаляет торренты со статусами `error`, `missingFiles` или `unknown`, очищая список от «битых» раздач.
* **Умная база данных**: Использует JSON-файл для отслеживания истории активности каждого торрента.
* **Гибкая настройка**: Все параметры (логин, пароль, лимиты, задержка запуска) вынесены в удобный файл `config.ini`.
* **Безопасный автозапуск**: Встроена задержка старта для корректной работы при одновременном запуске с Windows/qBit.
* **Подробные логи**: Ведет учет всех действий в `torrent_log.log` и автоматически удаляет логи старше 30 дней.

## Настройка (config.ini)

При первом запуске скрипт создаст файл конфигурации со следующими параметрами:
* `QB_URL`: Адрес веб-интерфейса qBittorrent (по умолчанию `http://localhost:8080`).
* `DAYS_LIMIT`: Сколько дней торрент должен пробыть в базе перед проверкой на удаление.
* `RATIO_THRESHOLD`: Минимальный прирост рейтинга, чтобы торрент считался «полезным».
* `LEECH_SUM_THRESHOLD`: Минимальная сумма личей за всё время, чтобы торрент не удалялся.
* `STARTUP_WAIT_SEC`: Пауза перед началом работы (полезно для автозагрузки).

## Как использовать

1. Скачайте готовую версию для вашей ОС из раздела **Releases**.
2. Разместите файл в удобном месте и запустите.
3. Отредактируйте появившийся `config.ini`, если ваши данные для входа отличаются от стандартных.
4. Добавьте ярлык программы в автозагрузку для полной автоматизации.
